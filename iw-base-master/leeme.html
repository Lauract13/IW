Proyecto

Consiste en informatizar el sistema actual de reservas para las pistas de voleibol de la UCM que tiene actualmente la univerdad

para ello existen 3 tipos de usuarios, un administrador que podrá crear a otros y realizar cualquier operación sobre las pistas

además del servicio principal que es reservar, este también podrá asignarle un equipo a un capitán, denominado Player; este

tipo de usuario puede hacer reservas todos los días de la semana, pero con un tope de 18 horas mensuales para los días entre

lunes y viernes. Y por último se encuentra el usuario Normal, que puede estar o no vinculado con la UCM (esto es para los descuentos)

el cual puede hacer reservas solo los fines de semana.
